Happy New Year
Hey team,
Cheers to a new year. Here's the latest from Design Systems:
In 2021, the Firefox Design Systems team goal was to focus on setting and documenting Firefox UI foundations within our team's design tool of choice, Figma.
The foundation of an application's user interface are all of the required elements used for building the complete user experience. Elements include color and typography styles, for example, and other "no-brainer" common components used to build typical interfaces, such as buttons and text inputs.
Last year, the Firefox foundation libraries released were:
  • Firefox Colors
  • Firefox Icons
  • Firefox Desktop Icons
  • Firefox Desktop Styles
  • Firefox Mobile Styles
  • Firefox Desktop Simple Components (Now deprecated)
New Year, New Release
Since our first release, we have been improving Firefox Desktop Components, and working on foundational styles for mobile apps. In this second release, we’re announcing a new Firefox Desktop Component library and a Firefox Mobile Styles library.
🎁 What’s New
  • Firefox Mobile Styles
    • iOS - Colors and Type.
    • Android - Colors and Type.
  • Firefox Desktop Components
    • This library is starting out with 19 components you can use to build UI for the Firefox Desktop Browser.
    • Button, Button Group, Checkbox, Context Menu, Divider, Dropdown, Form Field, Header, Icon Button, Label, Menu, Modal, Panel, Radio Button, Select, Site Preview, Split Button, Text input, and Toggle Switch.
🎉 Library Updates
  • Firefox Desktop Icons
    • Now with OS file type icons & new MacOS cursor options.
  • Firefox Desktop Styles
    • Had a non-breaking update to type styles for MacOS & Windows.
  • We deprecated Firefox Desktop Simple Components, and are releasing all components under one package, the Firefox Desktop Components library.
👀 What's next?
  • Growing Firefox Desktop Components.
  • Releasing Core Firefox UI components such as the Chrome under a new library, Firefox Desktop UI.
  • Mobile Component libraries for Android and iOS.
Year in Review...
🏆 Most used Figma library award goes to...Firefox
The foundation library with the most use has been Firefox Icons. Launched June 25, 2021, Firefox Icons has been used in files worked on by 14 distinct Figma teams.
🏆 Most used Figma component award goes to...the Search icon
The most used icon is the Search icon with over 2,000 instances used.
