🆘 Component Testing Crew: Looking for volunteers!
  • What?
    • We are looking for designers that want to join our crew to test component usability and functionality in Figma.
  • When?
    • 1-2 monthly Zoom sessions.
  • Why?
    • We want everyone in Firefox UX to give us feedback on how components in Figma work!
    • You will learn how to build Figma components and test their usability.
    • You will become an advocate for best component building practices.
    • You will be able to bring back Figma knowledge to your team!
  • Cool. How do I sign up?
    • Sign up here. Thanks :)

🎉 Library Updates
  • Desktop
    • Desktop Components v2.0.8
      • The latest version includes improved component usability. The only breaking change occurred inside dropdown lists. If you have ever used the Dropdown or Context Menu lists from the Desktop Components library, detach your current work before pulling in new changes.
      • While we avoid breaking changes, in the future, we will continue to communicate them if they are caused.
    • Desktop Styles v2.2.1
      • The latest version includes an improved Type scale.
  • Mobile
    • Android Components v1.0.0
      • The first release of Android Components includes App Bar Top and Buttons.
