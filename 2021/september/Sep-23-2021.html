Firefox Design Systems
Release 1.0
Hey team,
Today, we are excited to release more foundational libraries under the first version of our Design System. With this release you will now have access to Firefox Type, Icons, Colors, Shadows, and the first of our Desktop Simple Components. We’ve also built our first Fx Mobile Library, Icons! To have access to these libraries, join the Figma Firefox Design Systems team.
🎉 What’s New
  • Fx Mobile Icons
  • Fx Desktop Simple Components for both light and dark themes
